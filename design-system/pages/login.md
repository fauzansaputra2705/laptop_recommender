# Login Page — Design Override

## Deviation from MASTER

**Reason:** Login page uses landing-style hero split layout for brand continuity. Split-panel design (branding left, form right) matches modern SaaS patterns.

## Layout Adaptation
- Split-panel: 50% branding (hidden mobile) + 50% form
- Left panel: gradient background with floating blobs (matching landing hero)
- Right panel: centered form with social login
- Mobile: stacked, logo only at top

## Typography
- Headings: Space Grotesk 700 (from landing)
- Body: DM Sans 400/500 (from landing)
- Match landing's font choices for brand consistency

## Style Adaptation
- Same gradient + floating blobs as landing hero
- Card-style form container with subtle shadow
- Google button with brand styling (indigo on hover)
- Feature highlights with icon badges

## Features
1. **Social Login:** Google OAuth button with brand styling
2. **Info Box:** Explains Google account requirement
3. **Back to Home:** Link to landing page
4. **Terms:** Legal links at bottom

## Motion
- Same floating blob animations as landing
- Gradient shift animation
- Hover states on Google button (scale, shadow)
- Respects prefers-reduced-motion
