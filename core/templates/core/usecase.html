Activity Diagram — Alur Rekomendasi

flowchart TD
    A([User Login]) --> B{Metode Login}
    B -->|Email/Password| C[Autentikasi]
    B -->|Google OAuth| D[OAuth Callback]

    E -->|Gagal| B
    E -->|Berhasil| F[Dashboard]

    G --> H[Input Role & Budget]
    H --> I[Input Minimum Spesifikasi]

    J --> K[Validasi Input]
    K -->|Valid| L[Pick Cluster berdasarkan Role]
    K -->|Invalid| I
    L --> M[Normalisasi Preferensi dengan Scaler]
    M --> N[Hitung Cosine Similarity]
    N --> O[Filter: Budget & Minimum Spec]
    O --> P[Ranking Top-N]
    P --> Q[Generate Explain Result per Feature]
    Q --> R[Tampilkan Hasil Rekomendasi]
    R --> S{User Action}
    S -->|Lihat Detail| T[Detail + Breakdown Badges]
    S -->|Bandingkan| U[Compare View max 3]
    S -->|Export| V[Download Excel/PDF]
    S -->|Riwayat| W[Daftar Rekomendasi Sebelumnya]

Class Diagram — Model Utama

classDiagram
    class User {
        +id int
        +username str
        +email str
    }

    class Profile {
        +id int
        +user User
        +role str
        +is_admin() bool
    }

    class Brand {
        +id int
        +name str
    }

    class Processor {
        +id int
        +name str
        +brand str
        +cores int
        +clock_speed float
    }

    class Gpu {
        +id int
        +name str
        +brand str
        +vram_gb int
    }

    class Laptop {
        +id int
        +name str
        +brand Brand
        +processor Processor
        +gpu Gpu
        +ram_gb int
        +storage_gb int
        +price float
    }

    class ClusterModel {
        +id int
        +name str
        +is_active bool
        +scaler_params json
        +feature_order json
        +k int
    }

    class Cluster {
        +id int
        +model ClusterModel
        +cluster_index int
        +label str
        +interpretation str
    }

    class Preference {
        +id int
        +user User
        +role_target str
        +budget_min float
        +budget_max float
        +min_ram int
        +min_storage int
        +top_n int
    }

    class Recommendation {
        +id int
        +user User
        +preference Preference
        +selected_cluster Cluster
        +results json
        +precision_at_k float
    }

Sequence Diagram — Proses Rekomendasi

sequenceDiagram
    actor User
    participant Browser
    participant Django as Django Views
    participant Engine as Recommender Engine
    participant DB as Database

    User->>Browser: Buka /recommend/
    Browser->>Django: GET RecommendView
    Django->>DB: Ambil ClusterModel aktif
    DB-->>Django: ClusterModel + clusters
    Django-->>Browser: Render form preferensi
    Browser-->>User: Tampilkan form

    User->>Browser: Isi preferensi + Submit
    Browser->>Django: POST RecommendView
    Django->>Django: Validasi PreferenceForm
    Django->>Engine: generate_recommendation(preference, cluster_model)
    Engine->>DB: Ambil semua Laptop
    DB-->>Engine: Laptop queryset
    Engine->>Engine: Pick cluster berdasarkan role target
    Engine->>Engine: Normalisasi preferensi dengan scaler_params
    Engine->>Engine: Hitung cosine similarity
    Engine->>Engine: Filter budget & minimum spec
    Engine->>Engine: Ranking Top-N
    Engine->>Engine: explain_result per laptop
    Engine-->>Django: Hasil list[dict]
    Django->>DB: Simpan Recommendation
    DB-->>Django: OK
    Django-->>Browser: Render halaman hasil
    Browser-->>User: Tampilkan rekomendasi + breakdown
