# Landing Page — Design Override

## Deviation from MASTER

**Reason:** Landing page needs higher visual impact than internal app pages. Hero + bento grid pattern chosen for conversion focus.

## Style Adaptation
- Flat Design with subtle elevation on cards
- Blue primary (#3B82F6) + Cyan accent (#06B6D4) for brand, Emerald (#10B981) for CTA
- Light mode only (landing is public-facing, internal app handles dark)
- Bento grid for feature showcase

## Sections
1. **Hero:** Headline + subline + CTA
2. **Bento Grid:** 3 feature cards (clustering, recommendation, evaluation)
3. **How It Works:** 3-step process with icons
4. **CTA Section:** Login prompt for authenticated users

## Typography
- Headings: Space Grotesk 700
- Body: DM Sans 400/500
- Prices/specs: JetBrains Mono (from MASTER)
